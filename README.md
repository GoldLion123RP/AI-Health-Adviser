## Overview

Smart Health Adviser is an interactive web application designed to provide preliminary health advice based on user-reported symptoms. This tool helps users quickly understand potential health conditions and offers basic guidance while emphasizing the importance of professional medical consultation.

## Features

- **Interactive Symptom Selection**: Users can click on symptom cards to select their current health symptoms
- **Dynamic Diagnosis**: Intelligent algorithm that provides potential health condition suggestions
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Emoji-based Interface**: Engaging and user-friendly visual experience
- **Multiple Health Condition Detection**:
  - Dengue Fever
  - Malaria
  - Flu
  - COVID-19
  - Respiratory Infections
  - Allergies
  - And more...

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Responsive Web Design Techniques

## How It Works

1. Select symptoms by clicking on symptom cards
2. Click "Get Health Advice" button
3. Receive a detailed diagnosis with advice and potential health conditions
4. Use the "Reset" button to clear selections
